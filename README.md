# wuziqi

## 安装

```bash
$ git clone https://github.com/Snoire/wuziqi.git
$ cd wuziqi
$ sudo make install
```

## 卸载

```bash
$ sudo make uninstall
```

## 使用

```bash
$ wuzi
```

1. 下棋模式

   - 单击鼠标左键在棋盘放置棋子，玩家执黑子，计算机执白子

2. 调试 ~~（作弊）~~ 模式

   - 鼠标右键单独放置黑子

   - 鼠标中键单独放置白子

3. 重新开始

   - 双击右键
   - 双击中键
   - 滚轮上滑
   - 滚轮下滑
